# Native Web App Icons

This directory contains PWA icons and assets. In a production app, these would be properly designed icons.

For this demo, we're using placeholder references. In a real implementation, you would:

1. Create 192x192 and 512x512 PNG icons
2. Create maskable versions of the icons  
3. Add splash screen images for iOS
4. Create favicon files

The manifest.json references these files for PWA installation.
